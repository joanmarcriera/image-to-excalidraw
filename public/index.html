Local Diagram Extractor

Turn a diagram screenshot into an Excalidraw file.

Run this repo locally, upload an image, add your vision API details, and get back a downloadable .excalidraw output plus a browser preview.

Source: github.com/joanmarcriera/image-to-excalidraw

Input

No image selected yet.

Auto mode: SVG files are converted locally when possible. Raster images go through the vision provider you choose.

Output

Idle

Upload an image or load the bundled example, then generate a result.
